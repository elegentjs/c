# 简单git命令使用
git st  -> 查看当前文件状态

git add .  -> 将所有需要新增，更改，删除的文件纳入到git暂存区（stage）

git ci -m "提交信息" -> 执行提交，将stage中的文件提交至git

git push origin master -> 将本地库推送至远程git（github）

git pull origin master -> 从远程库拉取最新代码至本地库
